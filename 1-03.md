### Задание 2

Запустите терминал и выполните команды:

- echo $SHELL
- cat /etc/shells

*Ответ приведите в виде снимка экрана и прокомментируйте в свободной форме результаты выполнения указанных команд*

### Решение Задание 2

Команда 
```bash
echo $SHELL
```
выводит путь к текущей командной оболочке в операционной системе. В данном случае - это bash

Команда 
```bash
cat /etc/shells
```
выводит список путей к установленным командным оболочкам. 

Файл /etc/shells содержит список командных оболочек оболочек, которые можно использовать пользователями при входе в систему.
